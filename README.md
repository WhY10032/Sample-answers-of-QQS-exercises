# What is this?

This repository contains sample answers of *Quantitative Social Science: An Introduction (Kosuke Imai)* exercises in the format of R source code. *Quantitative Social Science: An Introduction (Kosuke Imai)* and *Quantitative Social Science: An Introduction in Tidyverse (Kosuke Imai)* are introductory level textbooks intended for undergraduate students majored in social science to acquire basic statistical knowledge and practical skills in data analysis using R.

This repository also includes datasets provided by the book for exercises. You can also download these datasets on https://press.princeton.edu/student-resources/quantitative-social-science for free.
